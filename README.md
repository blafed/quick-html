# BEFORE YOUR RUN

you need:
- linux
- bash
- cmake
- X11 (for GUI display)


# GET STARTED
1. to install
```
git clone https://github.com/blafed/quick-html
```

2. to run
```
cd quick-html
```
```
sh run
```

this should open new window showing html page.

edit this file, to show whatever html:

> test/res/d.html
